Simple Explorer
===========

Simple Explorer is an Open Source file explorer. There are inspirations from [CM Filemanager](https://github.com/CyanogenMod/android_packages_apps_CMFileManager/) and 
[PureFM](https://github.com/Doctoror/PureFM)
